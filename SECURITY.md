# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 2.x (current) | ✅ |
| < 2.0 | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in Aayurvani, do NOT open a public issue.

**Email:** marellatrivikrama@gmail.com

Include the following in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within **48 hours**.

If the vulnerability is confirmed:
- A fix will be prioritized immediately
- You will be credited in the release notes (unless you prefer anonymity)

If the report is declined, a clear reason will be provided.

## Scope

The following are in scope:
- Flutter app (iOS, Android, Web)
- Node.js backend API
- Database schema and queries
- Authentication and email flows

- Third-party dependencies
